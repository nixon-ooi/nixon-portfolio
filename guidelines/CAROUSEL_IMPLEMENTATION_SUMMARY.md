## ‚öôÔ∏è Technical Details

### Carousel Logic
```javascript
// Calculate position relative to current slide
let position = index - currentSlide;

// Handle looping
if (position < -3) position += totalCards;
if (position > 3) position -= totalCards;

// Assign classes based on position
if (position === 0) card.classList.add('center');
if (position === -1) card.classList.add('left-1');
// ... etc
```

### Auto-play System
- Interval: 5000ms (5 seconds)
- Pauses on: Carousel hover
- Resets on: Manual navigation (arrow, dot)
- Resumes on: Mouse leave

### Z-Index Layering
- **Center**: z-index: 10 (top)
- **Adjacent**: z-index: 5
- **Outer**: z-index: 4
- **Hidden**: z-index: 1
- **Navigation**: z-index: 20 (always on top)

## üé® Design Philosophy

### Why Carousel?
- **Engagement**: Interactive elements increase user interest
- **Space Efficiency**: Shows 6 cards in compact space
- **Focus**: Highlights one main card while showing context
- **Modern**: Contemporary UI pattern for portfolios

### Why 3-Card Focus?
- **Balance**: Not too few (limiting) or too many (overwhelming)
- **Preview**: Side cards provide browsing context
- **Clarity**: Clear hierarchy of importance

### Why Purple Accents?
- **Distinction**: Different from portfolio (which varies)
- **Consistency**: Unified critique page identity
- **Professional**: Purple associated with creativity
